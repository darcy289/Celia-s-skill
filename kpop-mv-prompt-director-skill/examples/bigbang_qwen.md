# K-pop MV 风格提示词包：BIGBANG 灵感 MV：高端街头

## 1. 风格识别
- 用户输入：BIGBANG 街头传奇
- 匹配风格：BIGBANG
- 匹配置信度：460
- 风格氛围：高端街头、华丽嘻哈、叛逆奢华
- 目标工具：qwen

## 2. 导演阐述
这支 MV 以“高端街头、华丽嘻哈、叛逆奢华”为核心视觉方向，把上传图片中的同一个人物包装成一个具有明确舞台身份的 K-pop 主角。画面会在霓虹夜店、LED地板、堆满钞票的赌场与豪华水晶吊灯大厅之间切换，用鱼眼镜头、慢动作爆炸、慢动作撒钱、爆闪灯光和霸气甩头、张开双臂建立完整的音乐录像带节奏。

## 3. 全局 Prompt
使用上传图片中的同一个人物作为【角色】，保持脸部身份一致；K-pop MV 质感，电影级灯光，高级商业影像，主体清晰，服装、妆造、场景与所选风格 DNA 保持一致。

## 4. Shot List / 分镜提示词
### 镜头1：开场世界观
- 时长：3-5s
- 功能：建立风格世界观与主角身份
- 千问 Prompt：大全景，镜头缓缓推进，上传图片中的同一个人物站在霓虹夜店、LED地板中央，慢动作撒钱、爆闪灯光环绕四周，整个画面弥漫着高端街头、华丽嘻哈、叛逆奢华的强烈气场。

### 镜头2：主角登场
- 时长：3-4s
- 功能：让上传角色正式进入 MV 空间
- 千问 Prompt：中景，上传图片中的同一个人物身穿皮草大衣、彩色发色、金属项链，从霓虹夜店、LED地板中缓缓走出，镜头带有轻微手持晃动，墨镜反光、金链晃动在光线中闪烁。

### 镜头3：道具特写
- 时长：2-3s
- 功能：用核心道具建立记忆点
- 千问 Prompt：特写镜头，钞票、香槟停落在上传图片中的同一个人物指尖，周围出现慢动作撒钱、爆闪灯光，柔光微距表现出高端街头、华丽嘻哈、叛逆奢华的氛围。

### 镜头4：第一段动作
- 时长：3-4s
- 功能：第一次展示角色动作与节奏
- 千问 Prompt：中全景，上传图片中的同一个人物完成霸气甩头、张开双臂，画面强调墨镜反光、金链晃动，镜头以鱼眼镜头、慢动作爆炸跟随，转场边缘出现慢动作撒钱、爆闪灯光。

### 镜头5：群舞/主舞台
- 时长：4-5s
- 功能：进入 MV 的主舞台段落
- 千问 Prompt：全景，上传图片中的同一个人物在霓虹夜店、LED地板中开始主舞台表演，低角度仰拍展现队形变换，动作以慢动作卡点，突出高端街头、华丽嘻哈、叛逆奢华。

### 镜头6：眼神钩子
- 时长：2-3s
- 功能：给观众一个强记忆点的面部特写
- 千问 Prompt：近景特写，上传图片中的同一个人物直视镜头，眼神凌厉，镜头极速推拉，画面中短暂出现慢动作撒钱、爆闪灯光，同时突出墨镜反光、金链晃动。

### 镜头7：副歌爆点
- 时长：4-5s
- 功能：第一次视觉高潮
- 千问 Prompt：全景，上传图片中的同一个人物完成霸气甩头、张开双臂，身后爆发慢动作撒钱、爆闪灯光，镜头环绕拍摄，形成高能副歌的第一次视觉高潮。

### 镜头8：权力低角度
- 时长：3-4s
- 功能：建立主角压场感
- 千问 Prompt：低角度仰拍，上传图片中的同一个人物自信舞蹈、举杯，皮草随动作甩动，身后的豪华水晶吊灯大厅不断闪烁，画面充满压场感。

### 镜头9：细节闪切
- 时长：2-3s
- 功能：用身体/妆造/符号细节制造质感
- 千问 Prompt：大特写，墨镜反光、金链晃动沿着上传图片中的同一个人物的身体或服装细节蔓延，镜头用微距捕捉纹理，并快速闪切到钞票、香槟。

### 镜头10：道具互动
- 时长：3-4s
- 功能：让道具参与叙事而不是只做装饰
- 千问 Prompt：中全景，上传图片中的同一个人物与钞票、香槟同框，完成自信舞蹈、举杯，镜头轨道横移，慢动作撒钱、爆闪灯光把人物和道具连接起来。

### 镜头11：第二空间展开
- 时长：4-5s
- 功能：从主场景切到第二空间，增加 MV 层次
- 千问 Prompt：远景，上传图片中的同一个人物在堆满钞票的赌场中自信舞蹈、举杯，动作以慢动作呈现，身后拖出残影，空间具有明显纵深。

### 镜头12：高潮快切
- 时长：3-5s
- 功能：把造型、动作、道具快速压缩成高能蒙太奇
- 千问 Prompt：快速剪辑蒙太奇，画面连续闪现：上传图片中的同一个人物完成霸气甩头、张开双臂、造型变化、钞票、香槟爆破、墨镜反光、金链晃动特写，以高频快切和闪白转场制造 MV 高潮。

### 镜头13：结尾定格
- 时长：4-6s
- 功能：用最终场景留下封面级尾镜
- 千问 Prompt：远景，上传图片中的同一个人物回到豪华水晶吊灯大厅，完成自信舞蹈、举杯，钞票、香槟停在肩头或身旁，镜头缓缓拉远，画面融入黑暗并留出 LOGO 位置。

## 5. 负面 Prompt
避免：文字水印、低清晰度、模糊面部、脸部身份漂移、畸形手指、畸形肢体、多余路人抢戏、廉价舞台、现代 UI 杂讯、过度塑料感、画面脏乱、过曝、过暗。

## 6. 使用建议
- 如果使用图生视频，把上传角色图作为首帧或角色参考图，并将【角色】替换为“上传图片中的同一个人物”。
- 如果模型一次只能生成短片，建议每个镜头单独生成 3-5 秒，再后期拼接。
- 如果要接近 60 秒，把镜头5、7、11、13拉长到 5-6 秒，其余镜头保持 3-4 秒。
