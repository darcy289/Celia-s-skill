# K-pop MV 风格提示词包：aespa 灵感 MV：元宇宙 AI

## 1. 风格识别
- 用户输入：aespa 风格
- 匹配风格：aespa
- 匹配置信度：300
- 风格氛围：元宇宙 AI、未来女战士、冷艳
- 目标工具：seedance

## 2. 导演阐述
这支 MV 以“元宇宙 AI、未来女战士、冷艳”为核心视觉方向，把【角色】包装成一个具有明确舞台身份的 K-pop 主角。画面会在流动的液态金属空间、数据流、无限延伸的数据隧道与液态金属王座之间切换，用快摇、环绕、极速推拉、液态金属溅射、故障条纹、数据粒子和大力舞蹈、猛地甩头建立完整的音乐录像带节奏。

## 3. 全局 Prompt
使用上传图片中的同一个人物作为【角色】，保持脸部身份一致；K-pop MV 质感，电影级灯光，高级商业影像，主体清晰，服装、妆造、场景与所选风格 DNA 保持一致。

## 5. 负面 Prompt
避免：文字水印、低清晰度、模糊面部、脸部身份漂移、畸形手指、畸形肢体、多余路人抢戏、廉价舞台、现代 UI 杂讯、过度塑料感、画面脏乱、过曝、过暗。

## 6. 使用建议
- 如果使用图生视频，把上传角色图作为首帧或角色参考图，并将【角色】替换为“上传图片中的同一个人物”。
- 如果模型一次只能生成短片，建议每个镜头单独生成 3-5 秒，再后期拼接。
- 如果要接近 60 秒，把镜头5、7、11、13拉长到 5-6 秒，其余镜头保持 3-4 秒。
