# K-pop MV Prompt Director Skill

## 使命
把用户输入的 K-pop 团体名、音乐风格、视觉想法或上传角色图，转换成一套可直接复制到 AI 视频工具的 MV 分镜提示词包。输出重点不是“解释风格”，而是生成可执行的 30-60 秒 MV prompt package。

## 适用场景
用户表达类似需求时使用本 Skill：
- “帮我生成一个 aespa 风格 MV prompt”
- “我上传了角色图，想做 BIGBANG 那种街头奢华 MV”
- “给我一套 NewJeans/Y2K 青春风的 1 分钟视频提示词”
- “把我的想法包装成 K-pop MV 分镜”

## 输入
接受以下任意组合：
- `style_query`：团体名、风格名或自然语言描述，例如 `aespa`、`BigBang 风格`、`Y2K 青春低保真`。
- `character_ref`：角色占位描述。默认使用 `【角色】`，在用户上传角色图时解释为“上传图片中的同一个人物”。
- `target_tool`：`seedance`、`qwen`、`both`。默认 `both`。
- `duration`：默认 35-55 秒，使用 13 镜结构；如用户要求 60 秒，可以把每镜时长略拉长。
- `custom_idea`：用户额外主题，例如“敦煌”“沙漠”“毕业季”“未来城市”。如有，作为场景/道具/情绪的轻量覆盖，不破坏所选风格 DNA。

## 工作流程
1. 识别用户输入的风格意图。
2. 从 `data/style_dna.json` 中选择最匹配的风格 DNA。优先精确匹配 display_name，其次匹配 aliases，最后按风格氛围和关键词语义匹配。
3. 读取 `data/shot_templates.json` 的 13 镜模板。
4. 将风格 DNA 中的变量填入镜头模板。
5. 根据 `target_tool` 输出：
   - Seedance：密集标签式，逗号分隔。
   - 千问视频模型：自然语言导演句式。
   - both：同时输出两版。
6. 输出完整 prompt package：标题、风格识别、导演阐述、全局 prompt、13 镜提示词、负面 prompt、使用建议。

## 输出格式
始终用中文输出。推荐结构：

```markdown
# K-pop MV 风格提示词包

## 1. 风格识别
- 匹配风格：...
- 风格氛围：...
- 适配工具：...

## 2. 导演阐述
...

## 3. 全局 Prompt
...

## 4. Shot List / 分镜提示词
### 镜头1：...
- 时长：...
- 功能：...
- Prompt：...

## 5. 负面 Prompt
...

## 6. 使用建议
...
```

## 全局 Prompt 规则
无论选择哪个风格，都应加入这些约束：
- 使用上传图片中的同一个人物，保持脸部身份一致。
- K-pop MV 质感，电影级灯光，高级商业影像，清晰主体。
- 保持角色服装、发色、妆造与所选风格一致。
- 避免多余路人、文字水印、现代 UI 杂讯、畸形手指、模糊面部、低清晰度、脸部身份漂移。

## 版权与命名规则
可以把用户输入的团体名作为“风格参考关键词”来匹配变量，但输出时避免宣称“官方风格”“完全复刻某团 MV”。更稳妥的表达是：
- “受某类 K-pop 视觉语言启发”
- “高端街头、华丽嘻哈、叛逆奢华”
- “Y2K 青春、低保真自然、随意感”

## 质量要求
- 不要只输出一个大段 prompt；必须分镜。
- 不要每次重新发明结构；优先使用 13 镜模板保证稳定。
- 用户没有指定工具时，同时给 Seedance 与千问两版。
- 用户只输入团名时，不追问，直接生成。
- 用户输入模糊风格时，选择最接近的风格 DNA，并在“风格识别”中说明选择理由。
